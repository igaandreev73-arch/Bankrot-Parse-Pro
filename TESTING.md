# Чек-лист тестирования Bankrot Parser Pro

## Функциональное тестирование

### Модуль парсинга (`parser.py`)
| Тест | Статус | Описание | Ожидаемый результат |
|------|--------|----------|---------------------|
| Запуск парсера без данных | ⬜ | Запуск `run_parser()` с пустой БД | Парсинг завершается, данные сохраняются |
| Повторный парсинг в тот же день | ⬜ | Запуск парсера, если сегодня уже был парсинг | Запрос подтверждения у пользователя |
| Отмена парсинга при ответе 'n' | ⬜ | Ответ 'n' на вопрос обновления | Парсинг отменяется с сообщением |
| Подтверждение парсинга при ответе 'y' | ⬜ | Ответ 'y' на вопрос обновления | Парсинг продолжается |
| Вывод количества новых записей | ⬜ | После сохранения данных | Вывод сообщения "Сохранено новых записей: X" |

### Модуль анализа (`analyzer.py`)
| Тест | Статус | Описание | Ожидаемый результат |
|------|--------|----------|---------------------|
| Анализ лота с кешем | ⬜ | Анализ лота, который уже есть в `analysis_cache` | Результат возвращается из кеша, `cached: true` |
| Анализ лота без кеша | ⬜ | Анализ нового лота | Вызов DeepSeek API, сохранение в кеш |
| Ошибка API DeepSeek | ⬜ | Неверный API ключ или недоступность API | Fallback-анализ, сохранение в кеш |
| Fallback-анализ | ⬜ | Проверка логики fallback | Корректные значения liquidity_score, risk_level |
| Валидация ответа API | ⬜ | Некорректный JSON от API | Обработка ошибки, переход к fallback |

### База данных (`database.py`)
| Тест | Статус | Описание | Ожидаемый результат |
|------|--------|----------|---------------------|
| Инициализация БД | ⬜ | Запуск `init_database()` | Создание таблиц `trades` и `analysis_cache` |
| Сохранение лота | ⬜ | Вызов `save_trades_to_db()` с корректным DataFrame | Данные сохраняются, возвращается количество |
| Защита от дублирования | ⬜ | Попытка сохранить тот же `lot_id` в тот же день | Дубликат игнорируется |
| Получение лотов с фильтрами | ⬜ | `get_trades_from_db()` с region и min_discount | Корректная фильтрация |
| Очистка старых записей | ⬜ | `cleanup_old_records(days_to_keep=7)` | Удаление записей старше 7 дней |

### REST API (`app.py`)
| Тест | Статус | Описание | Ожидаемый результат |
|------|--------|----------|---------------------|
| GET /api/trades | ⬜ | Без параметров | Список лотов, статус 200 |

## Инструкция по запуску тестов

### Установка тестовых зависимостей
```bash
pip install pytest pytest-asyncio httpx
```

### Запуск тестов
```bash
# Запуск всех тестов
pytest

# Запуск тестов конкретного модуля
pytest tests/test_parser.py

# Запуск с подробным выводом
pytest -v

# Запуск с покрытием кода
pytest --cov=.
```

### Ручное тестирование
1. **Локальный запуск:**
```bash
uvicorn app:app --host 0.0.0.0 --port 8000
```
Откройте http://localhost:8000

2. **Docker тестирование:**
```bash
docker-compose up --build
```

3. **CLI тестирование:**
```bash
python cli.py parse
python cli.py show --limit 5
python cli.py stats
```
